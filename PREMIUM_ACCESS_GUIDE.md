# Premium User Access Guide

## How to Access Premium Features

Premium users can access their exclusive features through multiple pathways:

### 📱 **Main Navigation Methods**

#### 1. **User Avatar Dropdown** (Top Right)
Click on your profile avatar in the top-right corner to access:
- **Profile & Settings** → Complete settings dashboard with tabs
- **My Articles** → Article management dashboard  
- **Analytics** → Premium analytics (Premium/Admin only)
- **Billing & Subscription** → Manage billing (Premium only)

#### 2. **Dashboard Homepage**
When logged in, you'll see:
- **Premium Features Highlight Section** (Premium users only)
  - Quick access buttons to Profile Settings, Billing, Notifications, Analytics
- **Enhanced Profile & Settings Card** with premium indicators
- **Analytics Card** (Premium/Admin only)

#### 3. **Direct URL Access**
Premium users can directly navigate to:
- `/dashboard/settings` → Main settings hub
- `/dashboard/settings?tab=profile` → Profile management
- `/dashboard/settings?tab=billing` → Billing & subscription
- `/dashboard/settings?tab=notifications` → Email preferences
- `/dashboard/settings?tab=security` → Security settings
- `/dashboard/analytics` → Analytics dashboard

### 🎯 **Premium Settings Features**

#### **Profile Tab** (`/dashboard/settings?tab=profile`)
- **Avatar Upload**: Change profile picture (JPG, PNG, GIF - Max 2MB)
- **Personal Information**: Update name, display preferences
- **Account Status**: View membership details, account type
- **Theme Preferences**: Light/Dark mode toggle

#### **Billing Tab** (`/dashboard/settings?tab=billing`)
- **Subscription Overview**: Current plan, next billing date
- **Payment Methods**: Credit/Debit cards, PayPal integration
- **Invoice History**: Download past invoices
- **Usage Statistics**: Monthly usage tracking
- **Subscription Management**: Upgrade, downgrade, cancel

#### **Notifications Tab** (`/dashboard/settings?tab=notifications`)
- **Email Preferences**: Choose notification types
- **Topic Subscriptions**: Get notified when articles are published on specific topics
- **Frequency Settings**: Daily, weekly, or instant notifications
- **Unsubscribe Options**: Granular control over email types

#### **Security Tab** (`/dashboard/settings?tab=security`)
- **Change Password**: Update account password
- **Two-Factor Authentication**: Enhanced security setup
- **Active Sessions**: View and manage logged-in devices
- **Security Logs**: Recent security activity

### 🔄 **User Experience Flow**

1. **Login** → See personalized dashboard with premium features highlighted
2. **Click Avatar** → Quick access to most-used features
3. **Navigate to Settings** → Comprehensive settings with tabbed interface
4. **Manage Subscription** → Easy billing and subscription management

### 💡 **Visual Indicators**

Premium users will see:
- ⚡ **Lightning bolt icons** next to premium features
- 👑 **Crown badges** for premium status
- 🎨 **Purple/blue gradients** on premium UI elements
- ✨ **Enhanced descriptions** explaining premium benefits

### 🔒 **Access Control**

- **Free Users**: Basic profile settings, limited features
- **Premium Users**: Full settings access, billing, advanced notifications
- **Admin Users**: All premium features + admin dashboard access

All premium features are automatically available once a user's tier is set to "premium" in the database.
